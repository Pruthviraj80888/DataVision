## 🎯 What Is DataVision Pro?

**DataVision Pro** is a fully reactive, single-file (`Dashboard3.html`) 
interactive analytics dashboard built with pure HTML, CSS, and JavaScript. 
Drop in any spreadsheet or data file and the dashboard **instantly** 
auto-detects your columns, computes real KPIs, builds 5 advanced 
visualizations, generates 7 AI-powered insights, and performs deep 
statistical analysis — all without ever sending your data to a server.

**Perfect for:**
- 📊 Data analysts who want quick visual insights
- 🏢 Business teams presenting KPIs without BI tools
- 🎓 Students learning data visualization
- 🔬 Researchers exploring datasets quickly
- 💼 Anyone who works with CSV or Excel files

## ✨ Features

### 📁 Data Upload
- ✅ Drag & drop file upload with animated progress bar
- ✅ Supports **CSV, Excel (.xlsx/.xls), JSON, TSV, TXT**
- ✅ Multiple file support
- ✅ **Full reset** when data is removed — zero leftover state
- ✅ All processing happens **in your browser** — data never leaves your device

### 📈 Auto-Generated KPIs
- ✅ Automatically detects **all numeric columns** in your file
- ✅ Displays up to **5 KPI cards** with real computed values
- ✅ Shows **trend direction** (↑ up / ↓ down) with percentage change
- ✅ Smart formatting: detects money ($), count, rate columns
- ✅ Animated **sparkline mini-charts** per KPI
- ✅ Displays total, average, min, max per metric

### 📊 5 Advanced Visualizations
| # | Chart | Description |
|---|-------|-------------|
| 1 | **Trend Line** | Primary metric over time/index (toggle Line ↔ Bar) |
| 2 | **Category Donut** | Breakdown by first categorical column (Donut/Pie/Polar) |
| 3 | **Performance Radar** | Multi-metric normalized comparison (current vs previous) |
| 4 | **Horizontal Bar** | Regional/group comparison across two metrics |
| 5 | **Multi-Metric Timeline** | All numeric columns on one timeline with toggle |

### 🧠 7 AI-Powered Insights
All insights are **100% computed from your actual data** — no hardcoded text:

| Insight | Method |
|---------|--------|
| 📈 **Trend Analysis** | First-half vs second-half comparison |
| 📐 **Statistical Profile** | Mean, median, std dev, skewness, CV |
| 🚨 **Outlier Detection** | IQR (Interquartile Range) method |
| 🔗 **Correlation Analysis** | Pearson r — finds strongest column pair |
| 🏆 **Category Ranking** | Top vs bottom performing groups |
| 🔮 **Trend Forecast** | Linear regression with 10-period projection |
| 🛡️ **Data Quality Report** | Missing values count + quality score (%) |

### 🔬 Deep Analysis Tabs
- **Insights Tab** — 7 auto-generated insight cards
- **Data Table Tab** — Sortable, paginated full data explorer + CSV export
- **Statistics Tab** — Distribution histogram + column comparison bar chart + stat boxes (mean, median, Q1, Q3, std dev, min, max, count)
- **Correlation Tab** — Color-coded heatmap matrix + scatter plot of strongest pair

### 🎛️ Interactive Controls
- **Filters** — Auto-built dropdown filters from your actual column values
- **Chart Type Toggle** — Switch chart types without reloading data
- **Timeline Toggle** — Show/hide individual metrics on multi-line chart
- **Column Selector** — Choose which column to analyze in Statistics tab
- **Sortable Table** — Click any column header to sort ascending/descending
- **Pagination** — Browse large datasets with page navigation
